# Cómo crear un producto en Finexa Cloud

1. En el menú principal, entra al módulo **Productos** para ver la lista de productos registrados.

2. Haz clic en el botón **Agregar** para crear un nuevo producto.

3. Escribe el nombre o descripción del producto y su código si utilizas códigos internos o de barra.

4. Completa los campos de precio (por ejemplo, precio al contado), costo y los datos de inventario que maneje tu sistema.

5. Selecciona la **categoría** y la **marca** o ID de marca, si aplica.

6. Define otras opciones como unidad de medida, impuestos, existencia mínima o cualquier parámetro adicional disponible.

7. Revisa que toda la información esté correcta y haz clic en **Guardar** para registrar el nuevo producto.
