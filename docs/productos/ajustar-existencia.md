# Cómo ajustar existencia de un producto

1. En el menú principal, entra al módulo **Productos**.

2. Ubica el producto y haz clic en el ícono de **tres puntos** de su tarjeta.

3. Selecciona **"Ajustar Existencia"**.

4. En la ventana **Gestionar Existencia**, elige el **Tipo de movimiento**: **Entrada** (sumar) o **Salida** (restar).

5. Escribe la cantidad a ajustar.

6. Haz clic en **Guardar** para aplicar el ajuste de inventario.
