# Cómo crear una Nota de Crédito

1. Ir al módulo de **CxC**.

2. Entrar a **Transacciones**.

3. Pulsar el botón de **Agregar**.

4. Seleccionar **Nota de Crédito**.

5. Llenar los datos: tipo de NC, cliente, valor de la NC, concepto, etc.

6. Selecciona las facturas o documentos a los cuales afectará dicha nota de crédito y el monto de cada uno en caso de ser varios.

7. Guardar.

:::warning
No se puede aplicar una Nota de Crédito si el cliente no tiene una factura o documento a afectar.
:::
