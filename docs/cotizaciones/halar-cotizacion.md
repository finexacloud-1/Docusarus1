# Cómo halar una cotización a factura

1. Ve a la ventana de **Factura** o **Pre-factura**.

2. Haz clic en el botón **"Más acciones"** que se encuentra a la derecha.

3. Selecciona **"Cargar cotización"**.

4. Aparecerá una ventana con todas las cotizaciones creadas y activas. Selecciona la deseada.

5. Si la operación se realizó correctamente aparecerá un mensaje que dirá "Excelente" y la cotización estará cargada.
