# Cómo eliminar un cliente en Finexa Cloud

1. En el menú principal, entra al módulo **Clientes** y ubica la tarjeta del cliente que quieres eliminar.

2. Haz clic en el ícono de **tres puntos** de esa tarjeta para abrir el menú de opciones.

3. Selecciona **"Eliminar"** en el menú desplegable.

4. Confirma la eliminación en la ventana de aviso para borrar el cliente de forma definitiva del sistema.
