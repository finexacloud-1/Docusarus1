# Cómo crear un recibo de CxC

1. En el menú principal, entra a **Ventas** y haz clic en **Recibo de pagos** para abrir el listado de recibos de ingresos.

2. Pulsa el botón **Agregar** para crear un nuevo recibo.

3. En la pantalla **Recibo de ingreso**, selecciona el **Cliente** al que vas a registrar el pago.

4. Verifica que el campo **Tipo** esté en **CxC** y elige la **Forma de pago**, **moneda** y otros datos como sucursal, referencia y cobrador.

5. Escribe el **concepto** y, si lo necesitas, una **nota** descriptiva del pago.

6. Confirma que el **Total neto** sea correcto según el monto que estás recibiendo.

7. Haz clic en **Guardar** para registrar el recibo de CxC en el sistema.
