# Cómo eliminar un recibo de ingreso

1. Una vez localizado el recibo (ver [cómo localizar un recibo](/docs/recibos/localizar-ver-recibo)).

2. Ve al campo de **acciones** y haz clic en los **tres puntos**.

3. Selecciona **Eliminar**.

4. Aparecerá una ventana de confirmación. Confirma que deseas eliminar el recibo de pago.
