# Cómo crear un recibo de apartado

1. En el menú principal, entra a **Ventas** y haz clic en **Recibo de pagos**.

2. Presiona el botón **Agregar** para crear un nuevo recibo de ingreso.

3. En **Cliente**, selecciona el cliente que hará el apartado.

4. En **Valor**, escribe el monto que el cliente está abonando.

5. Elige la **moneda**, la **forma de pago** y verifica la tasa de cambio si aplica.

6. En el campo **Tipo**, selecciona **Apartado** en la lista desplegable.

7. Completa los campos **Concepto**, **Sucursal**, **Referencia**, **Cobrador** y **Nota** si es necesario.

8. Revisa que el **Total neto** sea correcto y haz clic en **Guardar**.

:::note
Se debe crear el cliente primero para poder hacerle un apartado, por lo menos con los datos básicos.
:::
