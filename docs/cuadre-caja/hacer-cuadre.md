# Cómo hacer un cuadre de caja

Para realizar un cuadre de caja, este se divide en diferentes etapas:

## 1. Desglose de efectivo

Detalla el efectivo que tienes en caja. El dinero se desglosa en el campo **Cant.** según la cantidad y automáticamente en el campo de **Total** se sumará.

El **Campo para fondo** se utiliza de la misma manera que el efectivo en caso de que lo quieras hacer manual. De lo contrario, tenemos el botón **Calc. Fondo Aunt.** donde pones el fondo que corresponde a tu caja y el sistema se encarga de sacarlo en las denominaciones más pequeñas para las devueltas.

## 2. Tarjetas, transferencias, desembolsos y Backup

El flujo de dinero se integra al cuadre cuando vas al círculo con signo de **+** que se encuentra en la parte inferior derecha y vas introduciendo según la transacción que corresponda.

### 2.1. Desembolso de caja
Se utiliza para introducir las salidas de dinero de la caja cuando no se tiene caja chica.

### 2.2. Backups / Retiro de caja
Esta opción la puedes utilizar para sacar dinero de caja con una justificación o autorización con un fin específico.

### 2.3. Tarjeta
Aquí se registra el cierre de los verifones (pagos con tarjeta).

### 2.4. Transacción de depósitos
Registra los depósitos que se envíen al banco, ya sea como adelanto de caja, etc. Tienes la opción de adjuntar el depósito.

## 3. Vista de salida de efectivo registrada

En esta parte de la ventana podrás visualizar cada una de las transacciones que se le han hecho al día con la salida del efectivo.

## 4. Entregar

Una vez que tienes todos estos pasos realizados, presiona el botón **ENTREGAR**.

:::note
Cada vez que se le dé salida o se entregue dinero de la caja, el sistema pedirá la autorización del supervisor o encargado de caja.
:::
