# Cómo crear un Suplidor en Finexa Cloud

1. En el menú principal, haz clic en **Suplidor** para abrir el listado de clientes.

2. Presiona el botón **Agregar** para crear un nuevo cliente.

3. En **Datos del cliente**, escribe el **Nombre del suplidor** y su **RNC/Cédula**.

4. Completa los campos de teléfono/celular, dirección, ciudad/sector y correo electrónico.

5. Selecciona la **Sucursal** y, si aplica, la empresa u otros campos comerciales.

6. Agrega **notas** si necesitas guardar información adicional sobre el cliente.

7. Revisa los datos y haz clic en **Guardar** para registrar el nuevo cliente en el sistema.
