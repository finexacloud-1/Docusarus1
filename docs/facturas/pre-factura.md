# Cómo hacer una pre-factura

1. Ir al módulo de **Ventas**.

2. Seleccionar **Pre-factura**.

3. Una vez en la ventana de pre-factura, llena los campos requeridos para la facturación: cliente, tipo de empleado, condición de pago, RNC, vendedor, etc.

:::note
Recuerda que siempre se deben llenar las condiciones de la venta antes de buscar el producto.
:::

4. Guardar la factura.
