# Cómo localizar una factura

1. Ir al módulo de **Factura**.

2. Haz clic en **"Más acciones"**.

3. Selecciona **"Localizar factura"**.

4. Busca la factura en la barra de búsqueda, ya sea por número de factura, cliente, etc.

También puedes filtrar por fecha, contado o facturas activas.

5. Selecciona la factura.
