# Cómo eliminar (anular) una factura

1. Localiza la factura (ver [cómo localizar una factura](/docs/facturas/localizar-factura)).

2. En la parte superior derecha verás el icono del zafacón con la palabra **"Anular"**.

3. Aparecerá una ventana con un menú desplegable donde seleccionas la opción por la cual estás anulando dicha factura.

4. Una vez seleccionado el tipo de anulación, haz clic en **Anular**.
